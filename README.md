# Excel  

## Overview

This repository contains small Excel exercises and practice files created as I learn and improve my data analysis skills using Microsoft Excel. The files included here cover different Excel concepts such as formulas, functions, data cleaning, data validation, pivot tables, and basic analysis.

The goal of this repository is to document my learning journey and showcase practical Excel tasks that demonstrate how spreadsheet tools can be used for data analysis and problem solving.

## Skills Practiced

The exercises in this repository focus on developing skills in:

* Excel formulas and functions (SUM, IF, VLOOKUP, XLOOKUP, etc.)
* Data validation and error checking
* Data cleaning and formatting
* Pivot tables and summaries
* Basic data analysis techniques
* Dataset preparation for analysis
* Management tracking systems

## Repository Structure

Each file in this repository represents a small exercise or concept that I am studying. Examples include:

* Data validation exercises
* Creating calculated columns using formulas
* Email formatting and text manipulation
* Date validation and formatting
* Pivot table analysis

File names describe the topic or concept being practiced.
